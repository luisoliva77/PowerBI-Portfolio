A partir da tela inicial, é possível iniciar a exploração dos dados de duas maneiras:

🔎 Análise por fornecedor
Selecione um dos fornecedores exibidos para visualizar o desempenho das vendas relacionado exclusivamente a ele.

📊 Análise geral
Clique em Iniciar para acessar o dashboard completo com a visão consolidada de todos os fornecedores.
